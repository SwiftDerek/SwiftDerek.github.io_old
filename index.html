Ant Simulation


This project was the culmination of a class on Data Structures/Algorithms as part of my CS bachelor's which simulates an ant colony from beginning to end of execution. There is one queen ant and a multitude of scout, forager, and soldier ants each with a purpose of keeping the queen and colony alive. The colony have to ensure no enemy ants move to the same area as the queen while finding and transporting food to the queen. This was the biggest project I've had to work on and required many data structures to efficiently move the ants around the grid. It was coded with Java and used the Swing/AWT libraries to build the GUI.
